Intel(R) FPGA SDK for OpenCL(TM)(tm) version 20.3 README.txt
=================================================

This README.txt accompanies the Intel(R) FPGA SDK for OpenCL(TM) Release version 20.3.
OpenCL and the OpenCL logo are trademarks of Apple Inc., used by permission
of Khronos.

The Intel(R) FPGA SDK for OpenCL(TM) is a C-based heterogeneous parallel
programming framework for Intel(R) FPGA devices.
This is an implementation of the Khronos OpenCL 1.0 Specification.
This version passes the Khronos OpenCL Conformance
Testing Process. Current conformance status can be found at
http://www.khronos.org/conformance.

Please see the Intel(R) FPGA SDK for OpenCL(TM) Programming Guide for a summary
of supported features.

Although we have made every effort to ensure that this version
of the Intel(R) FPGA SDK for OpenCL(TM) software functions correctly, there may be
problems that we have not encountered. If you have a question or
problem that is not answered by the information provided in
the documentation, please find more support resources at
https://www.intel.com/content/www/us/en/programmable/support/support-resources.html.
